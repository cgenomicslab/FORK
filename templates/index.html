Reference Proteome Analysis Suite

Query and explore a local mirror of UniProt reference proteomes. Build phylogenetic trees, compute presence/absence matrices across organisms, construct fine-grained phylogenetic profiles from partitioned gene trees, and perform batch sequence and domain lookups.

Analysis Modules
Phylogenetics

Phylogenetic Tree

Fetch sequences by Pfam / taxon filters, align with MAFFT, infer a maximum-likelihood tree, and visualize interactively or export as a static figure.
Comparative Genomics

Presence / Absence

Build a protein-family presence/absence matrix across organisms, render a clustered heatmap, and drill into any cell for sub-profiles and domain architectures.
High-Resolution

High-Res Profile

Partition per-Pfam gene trees into subclades (paralog groups) and assemble a fine-grained phylogenetic profile that captures lineage-specific expansions.

Sequence & Domain Tools
